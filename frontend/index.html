Welcome! This wizard will help you set up your tracker in just a few steps. You'll validate your PokéWallet API connection and choose your preferred pricing and display settings.

Connect to PokéWallet

Your PokéWallet API key is configured in the .env file. Click the button below to verify the connection is working.

Set Your Preferences

Choose how you'd like the app to behave. You can always change these later in Settings.

Pricing Mode

Prices fetched from CardMarket when cards are added, and refreshed nightly. Enables portfolio value tracking.

No price fetching — track what you own and visualise set completion. Uses minimal API calls.

Grouped View Layout

How cards are arranged within each set section in grouped view.

Auto-load Full Sets

When opening a set, automatically fetch all cards from the API if the local cache is incomplete.

Set Card Images

Show card artwork for unowned cards in set detail. Your collection cards always show their image.

High API usage warning! With both Auto-load and Card Images enabled, opening a single large set can trigger hundreds of image proxy calls on top of the set fetch. On the free PokéWallet plan (100 calls/hour) this can exhaust your quota very quickly. Consider keeping one or both of these off until your collection is cached locally.

Settings
Pricing Mode Control whether market prices are fetched and displayed.
Full mode — prices are fetched from CardMarket when cards are added and refreshed nightly. Enables portfolio value tracking and P&L indicators.
Collection only — no price fetching. Track which cards you own and visualise set completion. Uses minimal API calls.
Heads up! Switching to full mode will fetch prices for all your cards and run nightly refreshes. This counts toward your API limit (80 calls/hour, 1,000/day on the free plan). Large collections may temporarily hit rate limits.
Grouped view layout How cards are arranged inside each set section when grouped view is active.
Auto-load full sets When opening a set, automatically fetch all cards from the API if the local cache is incomplete.
Heads up! Each set you open will use 1 API call to fetch its full card list. On the free plan (100 calls/hour) this can exhaust your quota quickly if you browse several sets. Incomplete sets are filled in automatically by an hourly background job — it may take a few hours if many sets need updating.
Set card images Show card artwork for unowned cards in set detail. Cards already in your collection always show their image.
